# Yabe logger

## 🔧 Install
```bash
npm i
```

❗ **Bot needs some environment variables:**

### Discord
```
DISCORD_CLIENT_TOKEN
DISCORD_SERVER_ID
DISCORD_CHANNEL_ID
```

You can get the different Discord ids with the discord [dev mode](https://discordia.me/en/developer-mode)

## 👨‍💻 Development
```bash
npm run dev
```

## 🖥️ Production
```bash
npm start
```
